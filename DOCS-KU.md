# 📘 بەڵگەنامەی کوردی بادینی — zm-langAi

> **نووسین:** کوردی بادینی (خەتی عەرەبی)
> **نامە:** zm-langAi — پلاتفۆرمێکی فێربوونی زمانی بە هوشمەندی دەستکاری

---

## ١. پووختەی پرۆژە

**zm-langAi** سیستەمێکی فێربوونی زمانییە کە بە هوشمەندی دەستکاری (AI) دروستکراوە. ئامانجی ئەم پرۆژەیە ئەوەیە کە فێربوونی زمان ئاسانتر، خیراتر، و شیاوتر بکات بۆ هەموو کەسێک — بەتایبەتی بۆ فێربوونی زمانی کوردی و دیالێکتەکانی (بادینی، سۆرانی، کرمانجی) و ئەرەبی و زمانە تریش.

ئەم سیستەمە بەسەر تەکنەلۆژیای نوێترینەوە دروستکراوە:

- **Node.js 24** و **TypeScript 5.9** بۆ سوورانەوەی جووڵانەی سیستەم
- **Express 5** بۆ ئاستی API
- **PostgreSQL** و **Drizzle ORM** بۆ ئاستی داتابەیس
- **Zod** بۆ پشتراستکردنی داتا
- **OpenAPI 3.1** بۆ قەرارداتی API
- **React** و **Vite** بۆ ئاستی پێشەکی (فرۆنت-ئێند)

---

## ٢. ئاکارەکانی ئێستا

### دامەزراندنی بنەمایی
- سیستەمی pnpm Workspace دامەزراوە بۆ بەڕێوەبردنی پرۆژە وەک مۆنۆریپۆ
- سێرڤێری API کار دەکات لەسەر Express 5 بە چۆچکردنی کامەڵی (logging)
- قەرارداتی OpenAPI نووسراوە و ئامادەیە بۆ پیشاندانی ئەندامانی API
- پەیوەندی داتابەیس ئامادەیە بە Drizzle ORM
- پشتراستکردنی داتا بە Zod v4

### ئامانجەکانی نزیک
- شێوازی دامەزراندن بۆ بەکارهێنەران (ناوتۆمارکردن، چوونەژوورەوە، بەڕێوەبردنی ئۆتۆریتە)
- دروستکردنی شێوازی فێربوونی بنەڕەتی (وشە، ڕستە، گوێستن و قسەکردن)
- دامەزراندنی سیستەمی گۆڕانکاری داتابەیس
- دروستکردنی ئاستی پێشەکی (فرۆنت-ئێند) بۆ بەکارهێنەر

---

## ٣. شێوازی ئەندازیاری

### ٣.١ بینینی گشتی

پرۆژە وەک مۆنۆریپۆ دروستکراوە. ئەمەش واتایە کە هەموو بەشەکان لەیەک شوێندا دەژین بەڵام جودا جودا کار دەکەن:

```
zm-langAi/
├── artifacts/         # کاری دەرەکی (سێرڤەر و فرۆنت-ئێند)
│   └── api-server/    # سێرڤێری Express 5
├── lib/               # کتێبخانە هاوبەشەکان
│   ├── api-spec/      # قەرارداتی OpenAPI (سەرچاوەی ڕاستی)
│   ├── api-zod/       # شێوازەکانی Zod (دروستکراوی خۆکار)
│   ├── api-client-react/ # هووکەکانی React Query (دروستکراوی خۆکار)
│   └── db/            # شێمای داتابەیس و Drizzle ORM
└── scripts/           # سکریپتە یارمەتیدەرەکان
```

### ٣.٢ دەستووری OpenAPI-پێشەکی

هەر گۆڕانکارییەک لەسەر ئاستی API دەبێت لە فایلی `lib/api-spec/openapi.yaml` دەستپێبکات. ئەم فایلە سەرچاوەی ڕاستی قەرارداتە. پاشان کۆددروستکردنەوە (codegen) بەخۆیەوە تایپی TypeScript، شێمای Zod، و هووکەکانی React Query دروست دەکات.

**ئەم فەرمانە بەکاربهێنە پاش هەر گۆڕانکارییەکی API:**
```bash
pnpm --filter @workspace/api-spec run codegen
```

### ٣.٣ ئامنیەتی گوندی هەڵوەشانەوە

سیستەمی pnpm Workspace دانراوە کە هیچ پەکێجێک کە کەمتر لە ١ ڕۆژ بەروارەکەی چاپکردنی هەیە نەیتوانرێت دامەزرێت. ئەمەش سیستەم دژی هێرشی `supply-chain` پارێزگاری دەکات.

## ٤. ئامانجی زمانی بادینی

بنەڕەتی ئامانجەکانی ئەم پرۆژە، پاراستن و فێرکردنی زمانی کوردی بادینییە. دیالێکتی بادینی زمانی نەتەوەیە بۆ کوردەکانی باشووری کوردستان (ناوچەی بادینان)، و هێز و جانداری ئەم زمانە بە یارمەتی تەکنەلۆژیا پاراستن پێویستە.

ئەم پرۆژە دەیەوێت:
- **فێرخانەی دیجیتاڵی کوردی** دروست بکات کە دیالێکتی بادینی بەتایبەتی تێدایە
- **زمانی کوردی لە سەرخۆیی خودی** فێرکردنی ئامادە بکات — بەبێ پاڵپشتی زمانی ئینگلیزی
- **کلتور و مێژووی کوردی** لەگەڵ زمان فێر بکات

## ٦. بینینی تەکنیکی

### ٦.١ سێرڤێری API

سێرڤێری API بەسەر Express 5 دروستکراوە لەگەڵ:

- **CORS** — ئامادەکراو بۆ پەیوەندیکردن لە دۆمەینی جیاواز
- **pino + pino-http** — تۆمارکردنی داتا لە هەموو ئازارەکانی داوا و وەڵام
- **Zod v4** — پشتراستکردنی داتای داوا و وەڵام بە تایپی TypeScript
- **هاوچەرخی خۆکار** — تایپەکانی TypeScript لە شێمای داتابەیس هاتن هەتا هووکەکانی فرۆنت-ئێند

### ٦.٢ ئەندامانی API

| ڕێگا | ئادرەس | وەسف |
|------|--------|------|
| GET | `/api/healthz` | پشکنینی تەندروستیی سیستەم |

### ٦.٣ داتابەیس

شێمای داتابەیس بەسەر **Drizzle ORM** دروستکراوە و هەر تەیبڵێک لەخۆیەوە دەبێت:
- تعریف تەیبڵی Drizzle
- شێمای دامەزراندن (insert schema) بە drizzle-zod
- تایپەکانی TypeScript (Insert و Select)

**فەرمانی گۆڕانکاری شێما:**
```bash
pnpm --filter @workspace/db run push
```

## ٨. دەستپێکردن

```bash
# ١. دامەزراندنی هەموو پەکێجەکان
pnpm install

# ٢. دانانی گۆڕاوەکانی ژینگە
cp .env.example .env
# ویرایشکردنی .env و دانانی DATABASE_URL

# ٣. هاتنی شێمای داتابەیس
pnpm --filter @workspace/db run push

# ٤. سەرهەڵدانی سێرڤێر
pnpm --filter @workspace/api-server run dev
```

## ٩. بەشداریکردن

هەر کەسێک بتوانێت یارمەتی بدات بەخێر بێت:

١. فۆرک کردنی ریپۆزیتۆری
٢. دروستکردنی شاخی تایبەت: `git checkout -b feature/ئامانجەکەت`
٣. گۆڕانکاری و پشتراستکردن: `pnpm run typecheck`
٤. کۆمیتکردن بە پەیامێکی ڕوونکەر
٥. کردنی Pull Request بەرەو `main`

**دەستووری کۆد:**
- لە سێرڤەردا `console.log` بەکار مەهێنە — لەبەرەوە بەکاربهێنە `req.log` یان `logger`
- هەموو گۆڕانکارییەکی API دەبێت لە `lib/api-spec/openapi.yaml` دەستپێبکات
- شێمای داتابەیسی نوێ دەچێت بۆ `lib/db/src/schema/`
